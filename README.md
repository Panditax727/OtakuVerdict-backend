# 🎌 OtakuVerdict - Backend (v1.0.0)

Bienvenido al núcleo de **OtakuVerdict**, una plataforma diseñada para gestionar reseñas y comparativas detalladas entre versiones de Anime y Manga. Este repositorio contiene la API robusta y escalable que alimenta el ecosistema del proyecto.

## 🚀 Tecnologías Utilizadas

El proyecto está construido sobre un stack moderno y eficiente:

- **Runtime:** [Node.js](https://nodejs.org/)
- **Lenguaje:** [TypeScript](https://www.typescriptlang.org/) (Tipado estático para mayor seguridad)
- **Framework Web:** [Express](https://expressjs.com/)
- **ORM:** [Prisma](https://www.prisma.io/) (Manejo de base de datos tipo "Type-safe")
- **Base de Datos:** [Supabase](https://supabase.com/) (PostgreSQL)
- **Gestor de Paquetes:** npm

## 🛠️ Características Principales

- **Arquitectura Escalable:** Estructura de carpetas organizada por servicios y controladores.
- **Base de Datos Relacional:** Modelos optimizados para manejar series, capítulos y comparativas de usuarios.
- **API REST:** Endpoints listos para ser consumidos por el frontend.
- **Integración Continua:** Configuración preparada para despliegues rápidos.
